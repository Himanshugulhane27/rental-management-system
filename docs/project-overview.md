# Project Overview

## System Requirements
- Property owners can add/manage properties
- Tenants can view available rentals
- Track rent payments and lease details
- Communication between owners and tenants

## Core Entities
1. **Properties**: address, rent, availability
2. **Tenants**: contact info, lease history
3. **Leases**: property-tenant relationship, terms
4. **Payments**: rent tracking, due dates

## User Roles
- **Property Owner**: Manage properties and tenants
- **Tenant**: View properties, make requests
- **Admin**: System oversight

## MVP Features
- Property CRUD operations
- Tenant registration
- Basic lease management
- Simple payment tracking
