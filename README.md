# Rental Management System

A modern rental property management application built with AWS serverless architecture.

## Features
- Property listing management
- Tenant management
- Lease tracking
- Rent payment monitoring

## Tech Stack
- **Backend**: Node.js, AWS Lambda, DynamoDB
- **Frontend**: React.js
- **Storage**: AWS S3
- **Deployment**: Serverless Framework

## Project Structure
```
rental-management-system/
├── backend/           # Lambda functions and API
├── frontend/          # React application
├── infrastructure/    # AWS setup documentation
└── docs/             # Project documentation
```

## Getting Started
1. Clone the repository
2. Install dependencies: `npm install`
3. Configure AWS credentials
4. Deploy: `npm run deploy`
